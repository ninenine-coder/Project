🚀 React i18n Demo

這是一個使用 React + react-i18next 的國際化演示

📋 功能特色：

  • ✅ 使用 react-i18next 實現中英文切換
  • ✅ Hero 區塊背景圖片輪播
  • ✅ 流程圖根據語言切換圖片
  • ✅ 五大情境模擬案例卡片
  • ✅ 響應式設計與動畫效果
  • ✅ 語言切換按鈕在右上角

要運行這個 React 組件，請執行：

npm install && npm run dev

🎯 組件結構說明

📁 文件結構

  • src/lib/i18n.ts - i18n 配置
  • src/components/PBLSKnowledgeBase.tsx - 主組件
  • src/main.tsx - React 入口
  • package.json - 依賴配置

🌐 國際化功能

  • 中英文翻譯資源
  • 語言切換按鈕
  • 圖片路徑動態切換
  • 本地存儲語言偏好

🎨 UI 特色

  • Framer Motion 動畫
  • 響應式網格布局
  • 漸變背景與陰影
  • 懸停交互效果
