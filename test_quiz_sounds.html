📋 音效流程說明

答題倒數音效：進入考試專區時播放（音量降低70%），播完後進入第一題

答題音效：每題開始時延遲0.5秒後循環播放（最大音量100%），直到選擇答案

正確音效：答對時播放（音量降低30%），停止答題音效

錯誤音效：答錯或超時時播放（音量降低30%），停止答題音效

完整流程：倒數音效 → 0.5秒延遲 → 答題音效 → 選擇答案 → 對錯音效 → 0.5秒延遲 → 下一題答題音效

✅ 視覺反饋說明

正確答案：綠色邊框 + 綠色文字 + 綠色勾勾圖標（✓）

錯誤答案：紅色邊框 + 紅色文字 + 紅色叉叉圖標（✗）

圖標位置：顯示在選項右側，垂直置中
