🔒 按鈕限制規則

📋 測驗進行中的按鈕限制

  • 允許的按鈕：中/英轉換按鈕、測驗選項按鈕
  • 禁止的按鈕：所有其他按鈕（側邊欄導航、用戶頭像、登出按鈕等）
  • 點擊效果：點擊禁止的按鈕會跳出警告視窗
  • 測驗完成後：所有按鈕恢復正常功能

🧪 測試檢查清單

📋 測驗進行中測試

  1. 開始測驗
    • 點擊「開始筆試測驗」進入測驗
    • 確認測驗界面正常顯示
  2. 測試允許的按鈕
    • 點擊中/英轉換按鈕，確認可以正常切換語言
    • 點擊測驗選項（A、B、C、D），確認可以正常選擇答案
  3. 測試禁止的按鈕
    • 點擊側邊欄「資訊」按鈕 → 應該跳出警告視窗
    • 點擊側邊欄「練習專區」按鈕 → 應該跳出警告視窗
    • 點擊側邊欄「歷史成績」按鈕 → 應該跳出警告視窗
    • 點擊側邊欄「團隊」按鈕 → 應該跳出警告視窗
    • 點擊用戶頭像 → 應該跳出警告視窗
    • 點擊登出按鈕 → 應該跳出警告視窗
  4. 警告視窗測試
    • 確認警告視窗顯示正確的中文訊息
    • 點擊「確認」→ 應該跳轉到考試專區
    • 點擊「取消」→ 應該繼續測驗

🏁 測驗完成後測試

  1. 完成測驗
    • 完成所有測驗題目
    • 進入測驗結果頁面
  2. 結果頁面按鈕測試
    • 點擊「返回考試專區」按鈕 → 應該正常跳轉
    • 點擊「重新測驗」按鈕 → 應該正常重新載入
  3. 恢復正常功能測試
    • 點擊側邊欄導航按鈕 → 應該正常跳轉
    • 點擊用戶頭像 → 應該正常顯示用戶選單
    • 點擊登出按鈕 → 應該正常登出
