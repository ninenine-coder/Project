🔧 測驗語言切換修復測試

🎯 問題描述

❌ 觀察到的問題

從截圖中可以看到：

  • ✅ 左側選單已經切換為英文（Information, Practice, Exam, History, Team）
  • ✅ 語言切換按鈕顯示為中文「中」
  • ❌ 題目和選項仍然是中文

問題原因：測驗狀態變量沒有正確暴露到全局作用域，導致語言切換器無法檢測到測驗正在進行中。

🔧 修復方案

✅ 核心修復點

1. 暴露測驗狀態變量到全局作用域：

// 在 exam_quiz.html 中添加
window.isQuizActive = () => isQuizActive;
window.isQuizCompleted = () => isQuizCompleted;
window.getCurrentQuestionIndex = () => currentQuestionIndex;
window.setCurrentQuestionIndex = (index) => { currentQuestionIndex = index; };
window.getSelectedQuestions = () => selectedQuestions;

2. 更新語言切換器中的狀態檢測：

// 在 language-switcher.js 中修改
const isQuizActive = typeof window.isQuizActive === 'function' ? window.isQuizActive() : false;
const isQuizCompleted = typeof window.isQuizCompleted === 'function' ? window.isQuizCompleted() : false;

if (isQuizActive && !isQuizCompleted) {
  window.handleQuizLanguageSwitch();
}

3. 修復變量引用：

// 使用正確的函數調用
const currentIndex = typeof window.getCurrentQuestionIndex === 'function' ? window.getCurrentQuestionIndex() : 0;
const questions = typeof window.getSelectedQuestions === 'function' ? window.getSelectedQuestions() : [];

// 正確設置題目索引
if (typeof window.setCurrentQuestionIndex === 'function') {
  window.setCurrentQuestionIndex(currentIndex + 1);
}

🧪 測試步驟

📋 詳細測試流程

  1. 準備測試環境
    • 確保有中英文題目文件（questions.json 和 questions_en.json）
    • 前往筆試測驗頁面
    • 開啟開發者工具 Console 標籤
  2. 測試測驗狀態檢測
    • 開始測驗
    • 在 Console 中輸入：window.isQuizActive()
    • 應該返回 true
    • 輸入：window.isQuizCompleted()
    • 應該返回 false
  3. 測試題目內容語言切換
    • 在測驗進行中點擊語言切換按鈕
    • 觀察 Console 中的調試信息
    • 確認題目文字從中文切換為英文
    • 確認選項文字從中文切換為英文
    • 確認可以正常回答切換後的題目
  4. 測試變量訪問
    • 在 Console 中輸入：window.getCurrentQuestionIndex()
    • 應該返回當前題目索引（數字）
    • 輸入：window.getSelectedQuestions()
    • 應該返回題目數組
    • 輸入：window.setCurrentQuestionIndex(1)
    • 應該成功設置題目索引
