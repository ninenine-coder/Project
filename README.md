# PBLS 專案：互動式醫療情境學習平台

## 專案簡介

PBLS 專案是一個旨在提供互動式醫療情境學習體驗的平台。它結合了筆試測驗、360 度實境教學影片和虛擬人模擬情境，幫助使用者在多種模式下學習和練習醫療急救知識。平台設計注重使用者體驗，提供清晰的介面、豐富的內容和即時的回饋機制。

## 主要功能

### 1. 筆試測驗 (Exam Area - Written Test)

*   **題庫擴充**：從 20 題擴充至 200 題，每次測驗隨機抽取 20 題。
*   **難易度標示**：每道題目顯示「易」、「中」、「難」的圖示及文字標籤，幫助使用者了解題目難度。
*   **即時回饋**：測驗結束後提供詳細的回饋頁面。
*   **動畫回饋**：當分數介於 0-20 分時，顯示一個獨特的「饅頭人」分層動畫，提供視覺化的鼓勵或提示。
    *   動畫包含身體、頭部、三條線和電池四個圖層。
    *   頭部進行上下移動動畫，三條線緊貼頭部右上角並隨頭部移動。
    *   電池放大並靠近頭部，進行閃爍動畫。
    *   整體動畫尺寸已放大，並調整了各元素的相對位置。

### 2. 練習專區 (Practice Area)

練習專區分為兩個主要部分：

#### a. 360 實境教學影片 (360 VR Teaching Videos)

*   **沉浸式學習**：嵌入 YouTube 360 度影片，提供身臨其境的教學體驗。
*   **RWD 支援**：影片播放器支援響應式設計，確保在不同裝置上都能正常播放。
*   **返回按鈕**：提供「返回練習專區」按鈕，位於頁面底部。

#### b. 虛擬人模擬 (Virtual Human Simulation)

*   **情境選擇**：透過下拉式選單選擇五種不同的醫療情境（心臟驟停、食物中毒、溺水、火災嗆傷、外傷）。
*   **詳細情境描述**：選擇情境後，顯示該情境的詳細描述和相關圖片。
*   **圖片展示**：情境圖片放大展示在左側，移除漸變背景框，使介面更簡潔。
*   **返回按鈕**：提供「返回練習專區」按鈕，位於頁面底部。

### 3. 歷史成績 (History)

*   **成績記錄**：記錄使用者過往的測驗和練習成績。
*   **毛玻璃風格**：頁面採用半透明毛玻璃卡片設計，與整體網站風格一致。

### 4. 團隊成員 (Team)

*   **成員介紹**：展示專案團隊成員資訊。
*   **毛玻璃風格**：頁面採用半透明毛玻璃卡片設計，與整體網站風格一致。

### 5. 全站通用設計 (Global Design)

*   **固定背景**：全站（除登入/註冊頁面外）採用一張固定的醫療主題背景圖片，位於最底層，不阻擋點擊。
    *   背景圖片已調整位置，整體下移 2 公分，確保畫面重心和頂部空間。
*   **毛玻璃卡片**：主要內容區塊和卡片（如測驗選項、練習選項、情境卡片、歷史成績卡片、團隊成員卡片）均採用半透明毛玻璃效果，讓背景圖片透出，提升視覺質感。
*   **透明背景**：所有主要容器的白色背景已替換為透明，以確保毛玻璃效果的正確呈現。
*   **標題對齊**：考試專區、練習專區、歷史成績和團隊成員頁面的主標題和副標題已調整，確保與頂部導航欄保持適當間距，並居中對齊。

### 6. 登入與註冊頁面 (Login & Register Pages)

*   **獨立背景**：登入和註冊頁面不使用全站固定背景，而是採用獨立的漸變背景。
    *   登入頁面背景已更新為與註冊頁面相同的淡紫色漸變。
*   **登入按鈕**：登入頁面的按鈕已修改為醒目的紅色漸變，並調整了懸停效果和陰影。
*   **毛玻璃風格**：登入和註冊容器也應用了增強的毛玻璃效果，保持簡潔專業的風格。

## 技術棧

*   **前端**：HTML5, CSS3, JavaScript
*   **資料**：JSON (用於題目管理)
*   **動畫**：CSS `@keyframes` 動畫, JavaScript (用於動態元素操作)
*   **圖片處理**：Python Pillow (用於生成動畫幀，儘管最終動畫改為 CSS/JS 實現)

## 功能特色

### 🎯 互動式學習體驗
- 多種學習模式：筆試測驗、360度影片、虛擬情境模擬
- 即時回饋機制：測驗結果、動畫鼓勵、詳細分析
- 難易度分級：題目難度標示，幫助學習者循序漸進

### 🏥 醫療情境模擬
- 五種情境：心臟驟停、食物中毒、溺水、火災嗆傷、外傷
