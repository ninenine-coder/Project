🔍 選項數量調試測試

🎯 問題描述

❌ 觀察到的問題

用戶反映："為什麼題目會出現8個選項"

  • ✅ 正常情況下，PBLS測驗應該是4個選項（A、B、C、D）
  • ❌ 但某些情況下會出現8個選項
  • ❌ 這可能導致用戶困惑和答題困難

可能原因：

  • 語言切換過程中選項被重複或合併
  • 題目文件格式問題
  • JavaScript邏輯錯誤導致選項數組異常

🔧 調試方案

✅ 已添加的調試功能

1. 在 displayQuestion() 函數中添加調試信息：

console.log('Original options count:', question.options.length);
console.log('Original options:', question.options);
console.log('Shuffled options count:', question.shuffledOptions.length);
console.log('Shuffled options:', question.shuffledOptions);

2. 在語言切換函數中添加調試信息：

console.log('New question options count:', newQuestion.options.length);
console.log('New question options:', newQuestion.options);

3. 修復選項標籤顯示問題：

// 支持更多選項的標籤
const optionLabels = ['A)', 'B)', 'C)', 'D)', 'E)', 'F)', 'G)', 'H)'];
const label = optionLabels[index] || `${String.fromCharCode(65 + index)})`;

📊 可能的原因分析

可能原因 描述 調試方法 狀態
語言切換問題 在語言切換過程中，選項被重複或合併 檢查 Console 中的語言切換調試信息 🔍 待確認
題目文件問題 questions.json 或 questions_en.json 中有異常題目 檢查題目文件格式和內容 🔍 待確認
數組操作問題 JavaScript 數組操作導致選項重複 檢查 selectedQuestions 數組內容 🔍 待確認
隨機化問題 shuffleOptions 函數有問題 檢查隨機化後的選項數量 🔍 待確認

🧪 調試步驟

📋 詳細調試流程

  1. 準備調試環境
    • 前往筆試測驗頁面
    • 開啟開發者工具 Console 標籤
    • 清空 Console 歷史記錄
  2. 測試正常情況
    • 開始測驗（不進行語言切換）
    • 檢查第1題的選項數量
    • 在 Console 中查看調試信息
    • 確認選項數量是否為4個
  3. 測試語言切換情況
    • 開始測驗後立即切換語言
    • 檢查切換後的選項數量
    • 在 Console 中查看語言切換調試信息
    • 確認選項數量是否異常
  4. 檢查題目文件
    • 在 Console 中輸入：fetch('data/questions.json').then(r => r.json()).then(console.log)
    • 檢查第1題的選項數量
    • 在 Console 中輸入：fetch('data/questions_en.json').then(r => r.json()).then(console.log)
    • 檢查第1題的選項數量
  5. 檢查數組狀態
    • 在測驗進行中，在 Console 中輸入：window.getSelectedQuestions()
    • 檢查返回的題目數組
    • 檢查每個題目的選項數量

🔍 預期調試信息

正常情況下應該看到的調試信息：

// 正常顯示題目時
displayQuestion: showing question 1: 在進行CPR時，正確的按壓深度應該是？
Original options count: 4
Original options: ["至少5公分", "至少3公分", "至少7公分", "至少10公分"]
Shuffled options count: 4
Shuffled options: ["至少7公分", "至少5公分", "至少3公分", "至少10公分"]
displayQuestion: created 4 options

語言切換時應該看到的調試信息：

// 語言切換時
Updating question 1 with new language version
New question options count: 4
New question options: ["At least 5 cm", "At least 3 cm", "At least 7 cm", "At least 10 cm"]
Updated entire selectedQuestions array with new language
Current question: What is the correct compression depth during CPR?
Current question options count: 4

如果出現8個選項，可能看到的異常信息：

// 異常情況
Original options count: 8
Original options: ["至少5公分", "至少3公分", "至少7公分", "至少10公分", "At least 5 cm", "At least 3 cm", "At least 7 cm", "At least 10 cm"]
// 或者
New question options count: 8
// 這表示選項被重複或合併了

📊 預期結果

✅ 正常情況下應該看到

  • 選項數量：每個題目都應該有4個選項
  • 選項標籤：A)、B)、C)、D)
  • 調試信息：Original options count: 4, Shuffled options count: 4
  • 功能正常：可以正常選擇和回答題目

🎯 如果發現異常

  • 記錄異常信息：記錄 Console 中的異常調試信息
  • 檢查觸發條件：確認是在什麼情況下出現8個選項
  • 檢查題目文件：確認題目文件格式是否正確
  • 檢查數組狀態：確認 selectedQuestions 數組是否正常
