🔄 測驗中語言切換邏輯測試

🎯 功能描述

✅ 智能語言切換邏輯

在測驗過程中進行中英文轉換時，系統會根據當前題目的作答狀態智能決定下一步行為：

  • 已作答題目：語言切換後直接進入下一題
  • 未作答題目：語言切換後繼續顯示當前題目（新語言版本）
  • 最後一題：如果是最後一題且已作答，直接結束測驗

🔄 邏輯流程圖

語言切換按鈕點擊
        ↓
檢查測驗是否進行中
        ↓
   是 → 檢查當前題目作答狀態
        ↓
   已作答 → 進入下一題（新語言）
        ↓
   未作答 → 繼續當前題目（新語言）
        ↓
   最後一題且已作答 → 結束測驗
        ↓
   完成語言切換

📋 測試場景

✅ 場景1：已作答題目

步驟：

  1. 開始測驗
  2. 回答第一題
  3. 點擊語言切換按鈕
  4. 觀察系統行為

預期結果：

  • 語言切換到英文
  • 直接進入第二題（英文版本）
  • 不顯示第一題的解釋

⏰ 場景2：未作答題目

步驟：

  1. 開始測驗
  2. 不回答第一題
  3. 點擊語言切換按鈕
  4. 觀察系統行為

預期結果：

  • 語言切換到英文
  • 繼續顯示第一題（英文版本）
  • 可以繼續回答該題

⚙️ 技術實現

核心函數：

// 檢查當前題目是否已作答
function isCurrentQuestionAnswered() {
  return userAnswers.length > currentQuestionIndex;
}

// 語言切換後重新顯示當前題目
async function displayCurrentQuestionAfterLanguageSwitch() {
  // 根據新語言載入題目文件
  // 更新當前題目內容
  // 顯示新語言版本的題目
}

// 測驗中的語言切換處理
async function handleQuizLanguageSwitch() {
  const isAnswered = isCurrentQuestionAnswered();
  
  if (isAnswered) {
    // 已作答 → 進入下一題
    currentQuestionIndex++;
    await displayCurrentQuestionAfterLanguageSwitch();
  } else {
    // 未作答 → 繼續當前題目
    await displayCurrentQuestionAfterLanguageSwitch();
  }
}

關鍵邏輯：

1. 語言切換觸發 handleQuizLanguageSwitch()
2. 檢查 userAnswers.length > currentQuestionIndex
3. 根據結果決定下一步行為
4. 重新載入新語言版本的題目
5. 更新界面顯示

🧪 測試步驟

📋 完整測試流程

  1. 準備測試環境
    • 前往筆試測驗頁面
    • 開始測驗
    • 確保有中英文題目文件
  2. 測試已作答題目場景
    • 回答第一題（選擇任意選項）
    • 等待顯示正確/錯誤答案
    • 在顯示解釋期間點擊語言切換按鈕
    • 確認直接進入第二題（新語言版本）
    • 確認不顯示第一題的解釋
  3. 測試未作答題目場景
    • 重新開始測驗
    • 看到第一題時不回答
    • 直接點擊語言切換按鈕
    • 確認繼續顯示第一題（新語言版本）
    • 確認可以正常回答該題
  4. 測試最後一題場景
    • 快速完成前面的題目
    • 到達最後一題並回答
    • 在顯示解釋期間點擊語言切換按鈕
    • 確認直接結束測驗並顯示結果
  5. 測試多次切換
    • 在測驗過程中多次切換語言
    • 確認每次切換都按照正確邏輯處理
    • 確認測驗進度正確
    • 確認最終結果正確
