🔐 註冊功能測試

📋 註冊表單功能

📧 電子郵件驗證

  • 格式驗證 (xxx@xxx.xxx)
  • 必填欄位檢查
  • 錯誤訊息顯示

🔒 密碼強度驗證

  • 大小寫英文 + 數字
  • 最少8個字符
  • 確認密碼一致性

👤 個人資訊

  • 姓名 (最少2個字符)
  • 電話/分機號碼
  • 必填欄位驗證

🏢 所屬單位選擇

  • 長庚大學 → 資管系、護理系、醫學系
  • 長庚科大 → 護理系
  • 長庚醫院 → 小兒科

🧪 測試步驟

  1. 打開註冊頁面：public/index.html
  2. 點擊「立即註冊」按鈕
  3. 填寫註冊表單：
    • 電子郵件：test@example.com
    • 密碼：Test123456
    • 確認密碼：Test123456
    • 姓名：測試用戶
    • 所屬單位：長庚大學
    • 部門：資管系
    • 電話：03-2118800#1234
  4. 提交表單並檢查驗證
  5. 確認註冊成功並跳轉

✅ 驗證功能

📱 響應式設計

註冊表單支援各種螢幕尺寸：

  • 桌面電腦 (1200px+)
  • 平板電腦 (768px - 1199px)
  • 手機 (320px - 767px)

表單會自動調整大小和佈局以適應不同設備。

🔧 技術實現

JavaScript 功能：

// 部門選項配置
const departmentOptions = {
    '長庚大學': ['資管系', '護理系', '醫學系'],
    '長庚科大': ['護理系'],
    '長庚醫院': ['小兒科']
};

// 密碼強度驗證
function validatePassword(password) {
    const hasUpperCase = /[A-Z]/.test(password);
    const hasLowerCase = /[a-z]/.test(password);
    const hasNumbers = /\d/.test(password);
    const minLength = password.length >= 8;
    
    return hasUpperCase && hasLowerCase && hasNumbers && minLength;
}
